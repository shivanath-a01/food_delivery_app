Food Delivery Backend System

A complete food delivery backend system developed using FastAPI, PostgreSQL, and SQLAlchemy ORM.
This project implements core food delivery workflows including authentication, restaurant management, food items, cart system, orders, payments, offers, reviews, delivery tracking, and admin analytics.
